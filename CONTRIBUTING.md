# 参与贡献

感谢你关注 `yingdao-rpa-console`。

当前公开仓重点面向前端产品体验与演示能力，最欢迎的贡献方向包括：

- 界面与交互优化
- 响应式与可访问性改进
- Mock 数据质量与演示完整度提升
- 文档完善
- 本地开发体验优化

## 开始之前

1. 先搜索是否已有类似 Issue。
2. 非小改动请先开 Issue 讨论。
3. 尽量让一次 PR 只解决一个明确问题。

## 本地开发

```bash
cd frontend/rpa-console-ui
cp .env.example .env.local
npm install
npm run dev
```

## Pull Request 要求

- 变更范围尽量聚焦
- UI 相关改动请附截图
- 说明你解决的实际问题，而不只是改了哪些代码
- 不要在公开仓引入私有后端假设
- 不要提交任何密钥、内部地址或测试凭据

## 提交前检查

请确认：

- `frontend/rpa-console-ui` 下 `npm run build` 通过
- Mock 模式仍可正常运行
- 如果改动影响对外行为，文档也同步更新

## 当前不接受的内容

- 索取私有后端源码
- 依赖内部基础设施才能运行的改动
- 引入真实生产配置或敏感信息的提交

## 需要帮助？

提 Issue 时建议附带：

- 你要解决的问题
- 预期行为
- 实际行为
- 复现步骤
- 截图或录屏
