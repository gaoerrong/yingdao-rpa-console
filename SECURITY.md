# 安全说明

## 支持范围

本公开仓库只包含：

- 前端页面与交互实现
- Mock 演示逻辑
- 对外文档与接口契约

因此适合提交的安全问题包括：

- 前端 XSS、信息泄露、权限绕过等问题
- 公开仓误提交了敏感信息
- 公开依赖存在明显安全风险

不适合提交到本仓库的内容包括：

- 私有后端实现问题
- 私有部署架构问题
- 不在本仓库中的内部服务安全问题

## 如何报告

如果发现敏感安全问题，请不要直接公开发 Issue。

建议：

1. 准备清晰的复现步骤
2. 说明影响范围与相关文件
3. 如有修复建议，可一并说明
4. 通过仓库维护者主页私下联系

如果问题属于“误提交敏感信息”，请第一时间反馈，不要继续传播或镜像相关内容。
