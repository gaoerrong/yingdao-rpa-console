# 行为准则

## 基本原则

我们希望这个项目保持一个务实、尊重、可协作的技术环境。

鼓励的行为包括：

- 提供基于事实的建设性反馈
- 尊重不同经验层次的参与者
- 明确区分“已验证事实”和“个人判断”
- 聚焦问题本身，而不是针对个人

不可接受的行为包括：

- 人身攻击、侮辱或嘲讽
- 骚扰、歧视性表达
- 恶意争论或反复刷屏
- 未经允许公开他人隐私信息
- 缺乏技术内容的大量无效 Issue / PR

## 适用范围

本准则适用于：

- Issues
- Pull Requests
- Discussions
- 仓库中的文档讨论

## 维护者处理方式

维护者有权对违反本准则的内容进行编辑、隐藏、锁定或拒绝合并。

## 举报方式

如果你遇到不当行为，且内容涉及敏感信息，请不要公开发帖，可通过仓库维护者主页私下联系。
